# Portfólio Acadêmico - Heloisa Cardillo

## Pré-requisitos
- Node.js
- MySQL

## Instalação
1. Clone o repositório
2. Instale as dependências
```powershell
npm install
```
3. Inicie o projeto
```powershell
npm run dev
```

O banco de dados será criado e populado automaticamente.
