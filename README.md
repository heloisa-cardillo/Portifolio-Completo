# Portfólio Acadêmico - Heloisa Cardillo

## Pré-requisitos

- Node.js
- MySQL
- Git

## Clonar Repositório

```bash
git clone https://github.com/heloisa-cardillo/Portifolio-Completo.git
cd Portifolio-Completo
```

## Instalação

```bash
npm install
```

## Configuração do Banco de Dados

1. Crie um banco de dados MySQL
2. Crie um arquivo `.env` na raiz do projeto
3. Configure as credenciais no `.env`

## Executar o Projeto

```bash
npm run dev
```

## Acesso

Abra no navegador:
`http://localhost:3000`
